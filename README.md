# SleeveSanctum

**Author:** Christian Ventura  
**Repository Purpose:** Final project for GIT417  

## Project Overview

SleeveSanctum is a single-page website designed to showcase premium trading card sleeves and provide an interactive number guessing game. The site emphasizes a retro terminal-inspired aesthetic while maintaining modern design within a fixed 1280px layout.

The website is built with HTML, CSS, and JavaScript, featuring:  

- A custom dark/light theme toggle
- Interactive product carousel displaying card sleeves  
- Number guessing game with match history  
- Terminal-style contact form with real-time feedback  

## Features

- Fixed 1280px width layout: Ensures consistent display across modern browsers, including Chrome and Safari  
- Retro-inspired UI: Terminal aesthetics, pixelated images, and stylized buttons  
- Accessible navigation: Smooth scroll to sections using anchor links  
- Dynamic content: JS-powered carousel, theme toggle, and game interactions  
- Contact form: Captures user input with immediate success or error messages  

## Technologies Used

- HTML5 – Semantic structure of the page  
- CSS3 – Grid and flex layouts, custom variables for themes, retro-style UI  
- JavaScript (ES6) – Theme toggle, carousel logic, number guessing game, form validation
